Background

Current tide pool apps list popular tide pools around Seattle, however, they do not tell the user when low tide is for the best exploration experience. In addition these apps while giving directions to the pools do not indicate there location relative to the users location. Ebb-Tracker provides all this information.

Development Problems

Using a GET request did not allow us to manipulate NOAA API data. To overcome this we created a proxy server. Originally Firebase was used as the database and host but because of Firebase hosting shortcomings we used Heroku to host.

Personal Contribution

Accessed NOAA API, parsing and reformatting into user friendly data chucks for addition to a package.json file using JavaScript.
